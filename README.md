# TntRun
## Description
A minecraft plugin to play TntRun.

## System requirements
- Works on 1.8.8-1.8.9 spigot/paper server.

## Main features
### Multiple games can function at the same time
- Games are split in separate worlds. (Max worlds 36 as the arena joining compass cannot hold any more items currently)
### All worlds have player protection
- Players can't break or place blocks.
- Player don't have hunger loss.
- Players can't take damage
- Natural mob spawning is off.
- Weather can't change.
### Commands
- tntrun (create | delete | disable | edit | enable | join | list | reload)
- spectator/spec/s
- leave/l/hub
- forceStart/fs
- fly
### Customisation
- Spawn location are set by setting world spawn.
- tntrun.yml to edit main settings.
  - World settings (e.g. control player damage).
  - Modify lobby item names.
  - Modify countdown lengths.
  - Modify max players.
  - Modify grace period and tnt remove delay.
  - Modify spectator item names.
- Scoreboard.yml to edit scoreboards with colour.
