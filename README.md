# TntRun
## Description
A minecraft plugin to play TntRun.

## System requirements
- Works on 1.8-1.8.9 spigot/paper server.
- Doesn't work with bungeecord.

## Main features
### Multiple games can function at the same time
- Games are split in separate worlds.
### Arenas have player protection
- Players can't break or place blocks
- Mobs can't spawn
- Weather can't change
### Customisation
- Spawn location are set by setting world spawn
- Can modify countdown time, force countdown time and max players
